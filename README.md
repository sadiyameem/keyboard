# Keyboard

# Description
- This keyboard is a way to help improve typing and accuracy by highlighting random keys.

# Features
- Highlight random keys
- Keypress detection

# How it works
1. A random key is highlighted (A-Z)
2. The user presses on the highlighted key on their keyboard
3. If correct:
    - A random key is selected
 
4. If incorrect:
    - User doesn't move on untill the correct key is clicked on
